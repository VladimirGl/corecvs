Detailed Description

template<typename _InternalTypeInput, typename _InternalTypeOutput = _InternalTypeInput, int stepSize = 1, int inputNumber = 1, int outputNumber = 1>
class corecvs::BaseAlgebraMulti< _InternalTypeInput, _InternalTypeOutput, stepSize, inputNumber, outputNumber >

This class describes the basic operations and function prototypes to work with input and output buffers.

Mostly it is consecutive iteration over the buffers.

Generally algebra is an interator + working type traits + operations with working types. This class implements interator functionality.
