Detailed Description

template<typename Type>
class corecvs::ObjectRef< Type >

This class is an smart pointer to the memory block.

It counts references to the block (counter is stored in a block itself)

This class can be used for two purposes. 1. Allocate a piece of memory with automatic reference count 2. Allocate an object with automatic reference count
