Detailed Description

Both essential and fundamental matrices hold the following property: 

             [ a_11   a_12  a_13 ]  [ x']    [ 0 ]
 [x, y ,1]   [ a_21   a_22  a_23 ]  [ y'] =  [ 0 ]
             [ a_31   a_32  a_33 ]  [ 1 ]    [ 0 ]

In case of Fundamental matrix is a point in the image captured by uncalibrated camera. In case of Essential Matrix is a point captured by normalized camera
