Detailed Description

This class holds the upper part of the triangular unitary matrix. 

    |  0   1   2   3   4   5   6 | 7
    |----------------------------|---
  0 |  x   0   1   2   3   4   5 |
  1 |      x   6   7   8   9  10 |
  2 |          x  11  12  13  14 |
  3 |       -      x  15  16  17 |
  4 |      | |         x  18  19 |
  5 |      | |             x  20 |
  6 |       -                  x |
    |-----------------------------
  7
   
For matrix of size NxN we will have to store (N-1)(N-2)/2 elements. (ie. for 7x7 matrix 7*6/2 = 21 element 0..20 )


To get the offset of the element at i,j (j>i) ((N-1) + (N - 1 - (i - 1))) / 2 * i + j - i - 1 = (2N - 1 - i) * i / 2 + j - i - 1


(i.e for element (3,4) -> (6 + 6 - 2) / 2 * 3 + 4 - 3 - 1 = 15 )
