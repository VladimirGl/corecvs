00024  *
00025  * The intrinsic correction parameters form
00026  * http://www.vision.caltech.edu/bouguetj/calib_doc/papers/heikkila97.pdf
00027  *
00028  * Here are the correction formulas
00029  *  \f{eqnarray*}
00030  *  \pmatrix{dx \cr dy} &=& \pmatrix{x \cr y} - \pmatrix{x_c \cr y_c} \\
00031  *                    r &=& \sqrt{dx^2 + dy^2} \\
00032  *                 r_{corr} &=& k_1 r^2 + k_2 r^4 \\
00033  *           \hat{r}_{corr} &=& \sum_{i=1}^{n} k_i r^i \\
00034  *
00035  *   \pmatrix{x_{coor}^t \cr y_{coor}^t} &=&
00036  *   \pmatrix{p_1 dx dy + p_2 (r^2 + 2 dx^2)\cr p_1 (r^2 + 2 dy^2) + p_2 dx dy} \\
00037  *
00038  *
00039  *   \pmatrix{x \cr y} &=&
00040  *   \pmatrix{x_0 \cr y_0} +
00041  *   \pmatrix{dx \cr dy} * r_{corr} +
00042  *   \pmatrix{x_{coor}^t \cr y_{coor}^t}
00043  *
00044  *   \f}
00045  *
00046  *   For Marquardt-Levenberg algorithm we will need derivatives of the function
00047  *
00048  *   for more details please read the code of getCorrectionForPoint() or read the Heikkila paper
00049  *
00050  */
