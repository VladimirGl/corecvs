Detailed Description

COOSparseMatrix is efficient matrix made in Matlab style - http://en.wikipedia.org/wiki/Sparse_matrix#Coordinate_list_.28COO.29.

Matrix 

  0 1 2 0
  3 0 0 4
  0 5 0 0

will be stored like elements = [1,2,3,4,5]
irow = [0,0,1,1,2]
icol = [1,2,0,3,1]
nphi = 5
