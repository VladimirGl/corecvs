00235 /**
00236  *  Matrix is searched in form
00237  *
00238  * \f[ P=\pmatrix{
00239  *        a_{00} & a_{01} & a_{02}\cr
00240  *        a_{10} & a_{11} & a_{12}\cr
00241  *        a_{20} & a_{21} &    1
00242  *        }
00243  * \f]
00244  *
00245  * We have a point \f$(x,y)\f$ and a line \f$ A u + B v + C = 0 \f$
00246  *
00247  * The matrix will transform the point \f$(x,y)\f$ to the \f$(u,v)\f$
00248  *
00249  * \f[ u = \frac {a_{00} x + a_{01} y + a_{02}} {a_{20} x + a_{21} y + 1} \f]
00250  *
00251  * \f[ v = \frac {a_{10} x + a_{11} y + a_{12}} {a_{20} x + a_{21} y + 1} \f]
00252  *
00253  * So we have minimize
00254  *
00255  * \f[ A u + B v + C = 0 \f]
00256  * \f[
00257  *  A \frac {a_{00} x + a_{01} y + a_{02}} {a_{20} x + a_{21} y + 1} +
00258  *  B \frac {a_{10} x + a_{11} y + a_{12}} {a_{20} x + a_{21} y + 1} + C = 0
00259  * \f]
00260  *
00261  * We will rewrite it as:
00262  *
00263  * \f[
00264  *  A ({a_{00} x + a_{01} y + a_{02}})  +
00265  *  B ({a_{10} x + a_{11} y + a_{12}})  +
00266  *  C ({a_{20} x + a_{21} y + 1}) = 0
00267  * \f]
00268  *
00269  *
00270  * \f[
00271  *  \pmatrix{
00272  *   Ax & Ay & A & Bx & By & B & C x & C y }
00273  *
00274  *  \pmatrix{ a_{00} \cr a_{01} \cr a_{02} \cr
00275  *            a_{10} \cr a_{11} \cr a_{12} \cr
00276  *            a_{20} \cr a_{21}   } =
00277  *
00278  *  \pmatrix{ -C }

00280  * \f]

00282  */
